# Darius Chiosa – HTML Portfolio

This repo is my personal portfolio built by hand in HTML, with no CSS or JavaScript. The goal is to show structure, semantics and basic frontend best practices, not just “make it look pretty”.

## What is this?

It’s a simple landing page where I pull together:

- A short profile summary, focused on aligning technical infrastructure with business ROI and FinOps.  
- Recent work experience (Impress, La Zambra, FlyVip Marketing).  
- BI, Excel/financial and statistical analysis projects with links to dashboards, models and academic work.  
- Skills, certifications and the tools I use or I am currently learning.  
- Education in Marketing, DAW/Computer Engineering and business/marketing master’s degrees.

Everything is built using plain HTML5: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`, accessible forms, `details/summary` for collapsible sections, plus some care on meta tags for SEO and social sharing.

## Project structure

- `index.html`: main portfolio page.  
- `images/`: profile photo, favicon and basic Open Graph assets.

No dependencies, no build, no frameworks. You just open `index.html` in the browser and that’s it.

## How to use

- Clone the repo or download the ZIP.  
- Open `index.html` in your browser.  
- If you want to adapt it, edit the text, links and image paths directly in the HTML.

## Why I built this

I come from a hybrid business–technical background and I use this project to show that, besides BI, data and marketing, I can also go down to the web layer and take care of the technical details from the ground up.
